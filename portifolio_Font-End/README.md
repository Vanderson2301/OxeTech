OxeTech

Meu portfólio pessoal desenvolvido para reunir meus projetos, tecnologias e experiências como desenvolvedor front-end.

A ideia do projeto é ter um espaço onde eu possa apresentar meu trabalho de forma simples, moderna e acessível, além de acompanhar minha evolução na programação.

🌐 Acesse: https://vanderson2301.github.io/OxeTech/

Sobre

O OxeTech nasceu da vontade de criar uma identidade própria para meus projetos e estudos. Nele você encontra informações sobre mim, minhas principais habilidades e alguns dos projetos que venho desenvolvendo.

O site foi construído com foco em responsividade, desempenho e uma interface agradável para diferentes dispositivos.

Executando localmente

Clone o repositório:

git clone https://github.com/Vanderson2301/OxeTech.git

Entre na pasta do projeto:

cd OxeTech

Abra o arquivo index.html ou utilize uma extensão como Live Server no VS Code.

Objetivo

Este projeto funciona como minha vitrine profissional e também como uma forma de colocar em prática conceitos de desenvolvimento web, design responsivo e boas práticas de front-end.

À medida que adquiro novos conhecimentos e desenvolvo novos projetos, o portfólio continuará sendo atualizado.
